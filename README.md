# 📊 Futures Spot-Futures Basis Crawler

一个用于抓取生意社期现数据的 Python 工具，支持**全历史数据获取 + 断点续爬 + 基差拆分**，适用于期货量化分析、基差套利研究等场景。

---

## 🚀 功能特点

* ✅ 自动抓取期现数据（现货 + 最近合约 + 主力合约）
* ✅ 支持 **断点续爬**（避免重复抓取）
* ✅ 自动识别有效交易日期
* ✅ 自动跳过不存在页面
* ✅ 拆分「现期差」为：

  * 差值
  * 百分比
* ✅ 输出结构化 CSV 数据
* ✅ 支持长期运行（可用于每日定时任务）

## 🧠 核心实现说明

### 1️⃣ 断点续爬

通过读取：

```
valid_dates.txt
```

获取最后一个日期，实现：

```text
增量更新（不会重复抓取）
```

---

### 2️⃣ 表格解析策略

由于网页结构为：

* 多级表头
* 交易所分段
* 单元格嵌套数据

程序做了以下处理：

* 跳过“交易所行”
* 跳过表头
* 只解析有效数据行
* 拆分「现期差」字段

---

### 3️⃣ 现期差拆分

网页中的现期差结构为：

```
15
0.01%
```

程序自动解析为：

```python
现期差_值 = 15
现期差_百分比 = 0.01
```
